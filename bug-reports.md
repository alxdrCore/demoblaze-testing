# Bug-reports demoblaze.com

## BUG-001 - Возможна оплата пустой корзины  
Модуль : Корзина
Серьёзность : Критично
Приоритет : Срочно

Описание : Возможна оплата корзины при отсутствии товаров в ней
Окружение : Windows 11, Mozilla 5

Шаги : 
1) Открыть сайт demoblaze.com
2) Авторизоваться на сайте
3) Перейти во вкладку "Cart"
4) Нажать на кнопку "Place order"
5) Заполнить поля формы оплаты
6) Нажать кнопку "Purchase"

Ожидаемый результат: Ещё на этапе нажатия кнопки "Place order" должна была появиться ошибка - "В корзине нет товаров для оплаты. Либо кнопка должна быть скрыта от пользователя"
Актуальный результат: Можно провести оплату пустой корзины

## BUG-002 - Втрина-колесо изображений на главном экране не ссылает на товар
Модуль : UI
Серьёзность: Критично
Приоритет: Критично

Описание: В колесе-витрине товаров на главном экране нельзя перейти по изображению товара на его карточку
Окружение : Windows 11, Mozilla 5

Шаги:
1) Зайти на сайти demoblaze.com
2) Зайти на главную страницу
3) Нажать на изображение товара в колесе-витрине

Ожидаемый результат: Переход на страницу товара
Актуальный результат: После нажатия на изображение товара ничего не происходит

## BUG-003 - Ответ от сервера код 200 при неудачной авторизации
Модуль: Авторизация
Серьёзность: Критично
Приоритет: Критично

Описание: При оправлении неправильных данных на авторизацию возвращается код 200 с сообщением "Wrong password" 
Окружение: Postman

Шаги:
1) Открыть Postman 
2) Запрос - Post, адрес - https://api.demoblaze.com/login
3) Headers : Content-type = application/json
4) Тело : Json - raw
{
    "username":"1",
    "password":"Mg=="
}
5) Нажать Send

Ожидаемый результат : Код 400 - Неправильно введены логин/пароль
Актуальный результат : Код 200 + сообщение "Wrong password"

## BUG-004 - Нет критериев для информационных полей при оплате
Модуль: Оплата товара
Серьёзность: Критично
Приоритет: Критично

Описание : При оплате товара в корзине можно ввести некорректные данные в важные поля (Страна, Город, Кредитная карта, Месяц, Год, Имя) и оплата успешно пройдет.  
Окружение: Windows 11 - Mozilla 5

Шаги : 
1) Зайти на сайт demoblaze.com 
2) Авторизоваться на сайте
3) Добавить любой товар в корзину
4) Зайти во вкладку “Cart” 
5) Нажать кнопку “Place order“
6) Ввести даннные в поля: 
“Name” - “123“,
”Country” - 123123,
”City” - 0003zalu,
“Credit Card” - “Aboba“,
”Month” - “amogus“,
”Year” - “isus”
7) Нажать кнопку “Purchase”
Ожидаемый результат : Появление сообщения об ошибке “Неверные данные в обязательном поле“
Актуальный результат : Успешная оплата

## BUG-005 - Нельзя перейти по изображению на карточке товара после использования кнопок навигации Prvious/Next
Модуль: UI
Серьёзность: Низкая
Приоритет: Низкий

Описание : Если находясь на главной странице нажать внизу кнопки навигации Previous/Next, то на открывшейся странице с карточками товаров нельзя будет по нажатию на изображение товара перейти на страницу с товаром.
Окружение: Windows 11 - Mozilla 5

Шаги : 
1) Зайти на сайт demoblaze.com 
2) Авторизоваться на сайте
3) Зайти на главную страницу мазагина
4) Пролистать вниз каталог товаров до кнопок Previous/Next
5) Нажать кнопку Previous или Next
6) Нажать на изображение на карточке любого товара
Ожидаемый результат : Переход на страницу товара при нажатии на изображение товара на карточке
Актуальный результат : При нажатии на изображение ничего не происходит. Функция не работает.
