# Demoblaze manual testing project

## О Проекте
Ручное и автоматическое тестирование для интернет-магазина электроники demoblaze.com

## Что сделано
- Составлен тест-план по сайту
- Написаны чек-листы для всех доступных модулей
- Проведено ручное тестирование
- Найдено и задокументировано 10+ багов
- Создана коллекция тестов в Postman для авторизации на сайте demoblaze 
- Составлены автоматические тесты на Python используя Selenium и Pytest для Xdist
- Автоматические тесты оптимизированы для удаленного тестирования через Docker + Selenium Grid

## Модули
- Авторизация и регистрация
- Каталог, фильтрация товаров, перемещение по каталогу
- Корзина
- Форма оплаты

## Инструменты
- Jira - баг-трекинг
- Firefox DevTools - анализ сетевых запросов
- Postman - API тестирование
- Python, Selenium, Pytest - для автотестов
- CI/CD - CI часть пайплайна для проверки наличия критических файлов
- Docker - для установки контейнеров для Selenium Grid
